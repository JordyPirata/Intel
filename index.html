¿Quiénes somos?

Intel es una empresa de tecnología estadounidense, fundada en 1968, que se dedica a la fabricación de procesadores para computadoras y dispositivos electrónicos. Lo cual nos posiciona cómo una de las empresas más grandes en la industria de semiconductores..

Giro de la empresa

El giro de la empresa es la producción y diseño de procesadores, tarjetas madre, dispositivos de almacenamiento, software y otros productos relacionados con la tecnología de la información. Sus procesadores se utilizan en una amplia variedad de dispositivos, incluyendo computadoras personales, servidores, dispositivos móviles, sistemas embebidos y sistemas de red. Además, Intel también se enfoca en la investigación y el desarrollo de tecnologías innovadoras en áreas como inteligencia artificial, automatización, seguridad y computación cuántica.

Java

Java has been one of the most popular programming language for many years. Java is Object Oriented. However it is not considered as pure object oriented as it provides support for primitive data types (like int, char, etc) The Java codes are first compiled into byte code (machine independent code). Then the byte code is run on Java Virtual Machine (JVM) regardless of the underlying architecture.
